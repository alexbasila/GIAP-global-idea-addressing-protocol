Hilfe

  • Android Chrome unterstützt keine Erweiterungen/Userscripts. Nutze Firefox (mit Tampermonkey), Kiwi Browser oder einen Desktop-Browser.
  • Bei einem Update in GitHub: Diese Seite neu laden → Install / Update. Tampermonkey zeigt dann Update an.
  • In Tampermonkey muss das Script auf chat.openai.com bzw. chatgpt.com aktiv sein.
